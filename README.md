# weiqi-go
极简手动挡下围棋，仿真下棋
---
可以仿真下棋(实际上就是不想精雕细沫，结果也还不错😂)
具体仿真体现在：
* 吃别人棋的时候必须一个个手动吃掉
* 下棋的位置任意，你甚至可以下在框里，就像现实里下棋一样（不是
* 如果下错了，可以回退，比如不小心下到框里了，不过不能回退到吃掉棋之前哦
* 最后结算也是自己一个个点目数的

### 注意
1. 下的过程不要随意调整大小，因为棋盘是相对定位的，棋子是绝对定位的，当然你也可以更改代码来使他们同步
2. 只是自己一时兴起，花了一个小时左右做的demo，潜在问题肯定很多，随便玩玩即可
